# Phase 5 Plan 01: Marketing Campaign Management

**End-to-end campaign management with segmentation, scheduling, and ROI tracking.**

## Overview

Vendors need to engage customers across channels. Plan 01 delivers campaign creation, audience segmentation, automated scheduling, and performance analytics.

## Task Breakdown

### Task 1: Schema & Types (15min)
- Campaign types: email, SMS, in-app
- Campaign status: draft, scheduled, sent, paused
- Metrics: sends, opens, clicks, conversions
- Segment criteria: channel, purchase history, recency

### Task 2: Engine (25min)
- createCampaign() with validation
- scheduleCampaign() for future execution
- executeCampaign() with batch sends
- trackMetrics() for performance data

### Task 3: UI (35min)
- CampaignList: table with filters, edit, delete, analytics link
- CampaignBuilder: step-by-step (name → template → segment → schedule)
- CampaignDetail: analytics dashboard with charts
- CampaignAnalytics: open rate, click rate, ROI

### Task 4: Routing (15min)
- Add /marketing-campaigns route
- Update sidebar navigation
- Add lazy import in App.tsx

## Acceptance Criteria

- AC-1: Create campaign with template, audience, schedule
- AC-2: Campaign executes on schedule (email/SMS)
- AC-3: Metrics tracked (sends, opens, clicks, conversions)
- AC-4: Analytics show ROI and performance trends
- AC-5: Build passes, UI responsive
