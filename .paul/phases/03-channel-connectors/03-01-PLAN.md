<objective>
## Goal

Establish multi-channel connector framework with fully functional Shopify integration, enabling vendors to authenticate with Shopify, import products, and sync inventory in real-time.

## Purpose

Channel connectors are the core value proposition — vendors need to pull product data from multiple e-commerce platforms into VendorFlow's centralized dashboard. Shopify is highest-priority channel. Framework established here will be extended to Amazon and WooCommerce in subsequent plans.

## Output

- Database migration adding channels table and extending products table for channel metadata
- Connector framework (base abstraction + interfaces for all future channels)
- Shopify OAuth authentication flow
- Shopify product sync implementation (fetch from Shopify API → insert/update in products)
- Initial UI for channel management (connect, disconnect, sync status)
</objective>

<task type="auto">
  <name>Task 3: Implement Shopify connector (auth + product sync)</name>
  <files>src/integrations/channels/shopify/shopify-auth.ts, src/integrations/channels/shopify/shopify-connector.ts, src/lib/channels.ts</files>
  <action>
    Extend BaseConnector for Shopify:
    
    shopify-auth.ts:
    - generateOAuthUrl(vendorId, redirectUri): string
      Returns Shopify OAuth consent URL with client_id, scopes, state (for CSRF)
    - exchangeAuthCode(code, state): Promise<AccessToken>
      POST to Shopify /oauth/access_token, return access token
    - storeShopifyAuth(vendorId, token, shopStore): void
      Store in channels table with platform=SHOPIFY, external_account_id=shop.myshopify.com
    
    shopify-connector.ts:
    - Class ShopifyConnector extends BaseConnector
    - getAuthUrl(): delegates to shopify-auth
    - handleAuthCallback(code, state): delegates to shopify-auth, stores result
    - fetchProducts(vendorId): Uses Shopify REST API v2025-01
      GET /admin/api/2025-01/products.json (with pagination: limit=250)
      Parse response, yield in batches
    - syncProducts(vendorId): 
      Loop through fetchProducts()
      For each product:
        - Transform: { name, price, image, inventory_quantity } from Shopify → local schema
        - Upsert to products table with (channel_id, external_product_id)
        - Update stock inventory
        - Handle partial failures (log error, continue)
      Return SyncResult { synced: N, errors: M, duration }
    - Error handling: Network errors, auth failures, malformed responses
    
    channels.ts (query layer):
    - getChannelByVendor(vendorId, platform): Promise<Channel | null>
    - storeChannel(vendorId, platform, externalId, token): Promise<Channel>
    - deleteChannel(channelId): Promise<void>
    - upsertChannelProduct(channelId, externalProductId, productData): Promise<Product>
    - getChannelStatus(channelId): Promise<ChannelStatus>
    - updateSyncStatus(channelId, status, error?): Promise<void>
    
    Avoid: Storing tokens in plain text — use Supabase Vault or pgcrypto
    Avoid: Blocking on sync — make it async/background job (future: Phase 4)
  </action>
